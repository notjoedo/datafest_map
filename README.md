## üí° Overview & Project Focus

The core objective of the project was to **Determine if affordability could affect prosperity**.

Our winning submission, the **DataFest Map Analysis Tool**, provided **an interactive way to visualize and compare the prosperity and affordability scores that we created based on the datasets we collected and cleaned**. It delivered novel, actionable insights through a combination of robust data processing and clear, effective visualization.
